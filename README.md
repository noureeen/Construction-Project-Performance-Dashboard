## Overview
This project is a **Project Performance Dashboard for the WIMS Hospital construction project**, built using **Power BI**.  
The dashboard brings together **schedule, cost, manpower, and critical path data** into one interactive view, helping project teams clearly understand performance and make informed decisions.

The focus of this project is not just reporting numbers, but **turning project data into meaningful insights**.

---

## Why This Dashboard?
Large construction projects generate a huge amount of data, but without proper visualization, it becomes difficult to track progress and identify risks.

This dashboard was created to:
- Monitor overall **project health at a glance**
- Compare **planned vs actual progress**
- Identify **critical and near-critical activities**
- Track **manpower deployment trends**
- Monitor **budget and value performance**
- Highlight **schedule risks early**

---

## What the Dashboard Shows

### Project Summary
High-level KPIs provide a quick understanding of the project scale and complexity:
- **Baseline Budget:** 690.00M  
- **Total Activities:** 4,880  
- **Critical Activities:** 499  
- **Near-Critical Activities:** 375  
- **Project Duration:** 24 months (729 days)

---

### Schedule & Progress Tracking
- Planned vs actual progress comparison to identify delays early
- Critical float analysis to focus on activities that directly impact project completion

---

### Milestone Tracking
Key milestones tracked with baseline finish dates:
- Mobilization completion  
- Mock-up rooms  
- Concrete slab completion  
- Long-lead item procurement  
- Testing, commissioning, and handover  

---

### Manpower Insights
- Monthly manpower trends showing workforce distribution across the project lifecycle
- Helps in understanding peak labor demand and planning resources efficiently

---

### Activity Duration Insights
- Identification of long-duration and high-impact activities
- Supports better planning, sequencing, and risk mitigation

---

### Cost & Value Trends
- Monthly planned values compared against cumulative project value
- Helps monitor spending patterns and value generation over time
